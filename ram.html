Random Access Memory (RAM)

Random Access Memory (RAM) is a type of computer memory that usually used to store working data, this means that the data is being actively required and changes and can be quickly disposed of when needed. RAM is volatile memeory, meaning the data on it is destroyed when power is lost such as a PC shutdown.
